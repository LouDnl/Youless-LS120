crontab (command will run at next startup):
edit crontab > sudo crontab -r
on first available line >  "@reboot xxxx", where "xxxx" is the command you wish to run.
save + exit

init.d (command will run at next startup): 
create /etc/init.d/start-youless.sh
contents > python3 /home/sb/scripts/liveview.py
chmod +x /etc/init.d/start-youless.sh

rc.local (command will run at next startup):
edit /etc/rc.local
add command

systemctl service (command will run when you start the service):
create service file > /etc/systemd/system/youless.service
contents:
[Unit]
Description=Youless Energy service
After=network.target
StartLimitIntervalSec=0
[Service]
Type=simple
Restart=always
RestartSec=1
User=root
ExecStart=/usr/bin/python3 /home/sb/scripts/liveview.py

[Install]
WantedBy=multi-user.target

Then reload the service files:
sudo systemctl daemon-reload
start the service > sudo systemctl start your-service.service
enable the service on reboot > sudo systemctl enable your-service.service
